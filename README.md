# ai-rest-service

## Installation

1. Install Python
2. Install Pip: ```sudo apt install python3-pip```
3. Install Flask: ``sudo apt install python3-flask``
4. Run package script: ```pip install -r requirements.txt```
5. Restart your IDE

## Usage
Run with: ``flask run``

## Remove all Packges
```pip uninstall -r requirements.txt -y```
